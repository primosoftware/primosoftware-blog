Finder

Show hidden files:

defaults write com.apple.finder AppleShowAllFiles YES

Hold the ‘Option/alt’ key, then right click on the Finder icon in the dock and click Relaunch

Homebrew

Git

Git LFS

Install

Install via Homebrew:

brew install git-lfs
git lfs install

Close/Reopen Terminal and SourceTree GUI client after Git LFS installation.

Configure storage directory (optional)

By default the files will be stored under each repository, e.g. avblocks-test\.git\lfs. You can configure another location with git config.

git config --global lfs.storage ~/GitLargeFiles

Development Tools

Xcode

Install Command Line Tools (CLT) for Xcode:

xcode-select --install

If you get:

xcode-select: error: command line tools are already installed, use "Software Update" to install updates

Try this:

softwareupdate --all --install --force

If that doesn’t show you any updates, run:

 sudo rm -rf /Library/Developer/CommandLineTools
 sudo xcode-select --install

gmake

Install latest GNU Make via Homebrew:

brew install make

CMake

Install via Homebrew:

cmake --version
